# DATA ANALYTICS CAPSTONE PROJECT

A simple, modular project to learn Excel, SQL, Python, and Power BI.

Each module takes 1-2 hours.

## The Four Modules

**Module 1: Excel**
Clean sales data, create pivot tables, and make simple charts.

**Module 2: SQL**
Create database, write queries with views, and analyze data.

**Module 3: Python**
Load data, create visualizations with Seaborn, and build a Streamlit dashboard.

**Module 4: Power BI**
Import data, create charts, and add interactive filters.

## Choose Your Path

**Path 1: Complete Journey**
Do all 4 modules in order. Total time: 4-8 hours

**Path 2: Python and Power BI Only**
Skip Excel and SQL. Start with Module 3. Total time: 2-4 hours

**Path 3: Just One Module**
Pick any single module to practice that tool. Total time: 1-2 hours

## What You Need

**Data File:**
sales_data.csv (provided)

**Software:**
- Microsoft Excel (Module 1)
- PostgreSQL with DBeaver (Module 2)
- Python with pip (Module 3)
- Power BI Desktop (Module 4)

## Getting Started

1. Download sales_data.csv
2. Open MODULE_1_EXCEL.md (or whichever module you're starting with)
3. Follow the instructions
4. Complete the tasks

## The Data

Sales data for 2024:
- 8 products (Laptop, Phone, Tablet, Headphones, Mouse, Keyboard, Monitor, Charger)
- 4 regions (North, South, East, West)
- Approximately 1,200 transactions across 12 months

## What You'll Learn

**Excel:** Data cleaning, pivot tables, and charts

**SQL:** Database design, queries with GROUP BY, and creating views

**Python:** Streamlit dashboards, Seaborn visualizations, and pandas data manipulation

**Power BI:** Data import, DAX measures, and interactive visuals

## Module Outputs

**Module 1:** sales_analysis.xlsx (cleaned data with analysis)

**Module 2:** sales.sql (database setup and queries)

**Module 3:** app.py (Streamlit dashboard application)

**Module 4:** sales_dashboard.pbix (Power BI dashboard file)

## Time Breakdown

Each module is designed to take 1-2 hours:
- Setup and data loading: 10-15 minutes
- Main tasks: 45-75 minutes
- Review and polish: 10-15 minutes

You can complete one module per session or all four over a weekend.
